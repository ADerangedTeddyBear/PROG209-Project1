Welcome to the Videogame review page!

This is a JQuery Mobile single-page web application created by Thai Anh Van Vu and Darius Cooper for PROG 209. This web application revolves around users submitting reviews for video games and gaming systems featuring a 1-10 scale as well as a section for comments.

Users can also browse a list of reviews that have been added with games and systems being in separate columns. Users can also click on items on the list to go to a separate page that displays the selected review.

There is also a page for the most recent game review and system review that changes as reviews are added.

Thank you for visiting and we hope you enjoy.

Created by Thai Anh Van Vu and Darius Cooper
